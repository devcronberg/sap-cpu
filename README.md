![](Billeder/sap-cpu-v1.png)

## Version 2

![](Billeder/sap-cpu-v2.png)

Læg tal fra RAM sammen

- Tal i RAM på adresse 0 og 1
- 0 i input
- Input Out + MAR In
- Ram Out + A In
- 1 i input
- Input Out + MAR In
- Ram Out + B in
- ALU Out + Output In
